Key Concepts

The World Wide Web (WWW)

The internet is a network of computers communicating with each other, establishing links that make up the 'web'. The key actors involved in this web are:
the client that has a web 'browser' such as Google Chrome or Firefox, the internet itself, the communications protocol - for instance HTTP, and servers that serve content such as HTML documents along with images, videos and so on.

How Computers Process Commands

Computers process instructions literally, meaning they can't make handle multiple inferences of a statement (unless they are trained to do so).
This implies that computers can't deal with ambuiguity and hence are very sensitive to typos and mistakes.

HTML: The Basics

Tags, Elements, Attributes

Tags are the building blocks of HTML. Almost every new idea in HTML is enclosed in opening and closing tags, with the exception of certain self closing tags such as img and br.
Attributes are properties of HTML tags that may or may not have values and specify details of how the element should behave. Elements can be understood as a complete html idea or tag, having independent existence in the DOM.

Inline and Block Elements

The core idea is that block elements such as 'p', 'div', 'form' have an independent box or container that creates a line break, while inline elements such as 'a', 'img' allow you to continue on the same line.

It may be noted that 'br' is an inline element, although it may appear to behave as block.
